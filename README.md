# AIcp# 智能屏幕录制程序

AIcp# 是一个纯本地运行的专业屏幕录制程序，专注于隐私保护和高性能。所有数据处理都在本地完成，无需网络连接。

## ✨ 主要特性

- **🎯 定时录制**: 精确设置录制时长，自动停止录制
- **🖥️ 多屏支持**: 支持选择不同屏幕进行录制
- **🎨 现代化UI**: 直观美观的用户界面，操作简单便捷
- **🔒 完全本地化**: 零网络依赖，所有数据处理在设备上完成
- **🛡️ 增强隐私保护**: 敏感操作本地处理，可选文件加密，权限透明化
- **⚡ 资源高效**: 精简安装包，智能资源管理，硬件加速最大化
- **🌐 跨平台支持**: Windows、macOS、Linux支持
- **🚀 高性能**: 硬件加速编码，优化的内存管理

## 🎬 核心功能

### 定时录制功能
- **精确时间控制**: 支持设置小时、分钟、秒的组合时长
- **智能提醒**: 定时结束前2秒自动恢复窗口显示
- **实时倒计时**: 显示录制剩余时间
- **自动停止**: 到达设定时间后自动停止录制

### 录制设置
- **多种帧率**: 支持24/30/60 FPS选择
- **自定义路径**: 可选择保存位置和文件名
- **屏幕选择**: 支持多显示器环境下选择录制屏幕
- **自动最小化**: 录制时可选择自动最小化窗口

### 用户界面
- **分栏布局**: 左侧控制面板，右侧状态显示
- **状态指示**: 颜色编码的状态提示
- **图标美化**: 直观的表情符号图标
- **响应式设计**: 支持窗口大小调整

## 技术架构

### 分层设计

1. **用户界面层**: 基于Qt 6的跨平台GUI
2. **应用服务层**: 业务逻辑和服务管理
3. **核心引擎层**: 音视频处理和编码
4. **平台抽象层**: 屏幕捕获、音频捕获和硬件编码的平台特定实现

### 关键技术

- Qt 6.4 (GUI + 基础库)
- FFmpeg 6.0 (LGPL)
- 平台专用API (DXVA/NVENC/VideoToolbox)
- SpeexDSP + RNNoise (音频处理)
- SQLite3 (本地任务存储)

## 构建

### 环境要求

- C++17 或更高版本
- Qt 6.4 或更高版本
- CMake 3.16 或更高版本
- FFmpeg 6.0 开发库

### 构建步骤

```bash
# 克隆仓库
git clone https://github.com/yourname/AIcp#.git
cd AIcp#

# 创建构建目录
mkdir build
cd build

# 配置项目
cmake ..

# 编译项目
make -j$(nproc)

# 运行程序
./AIcp#
```

## 🚀 快速开始

### 基本录制
1. 启动AIcp#应用程序
2. 配置录制设置（输出路径、帧率、屏幕选择）
3. 点击"🔴 开始录制"按钮
4. 完成录制后点击"⏹️ 停止录制"按钮
5. 在指定位置查看录制的文件

### 定时录制
1. 勾选"启用定时录制"
2. 设置录制时长（小时、分钟、秒）
3. 点击开始录制
4. 系统会在设定时间后自动停止录制
5. 录制结束前2秒窗口会自动恢复显示

详细使用指南请参阅：[定时功能使用指南](docs/timer_feature_guide.md)
